# Matemática Discreta

Repositório contendo exercícios, listas, implementações e materiais desenvolvidos durante a disciplina **Matemática Discreta** da Universidade Federal de Alfenas (UNIFAL-MG).

## 📚 Sobre a disciplina

A disciplina de Matemática Discreta fornece a base matemática para diversas áreas da Ciência da Computação, como algoritmos, estruturas de dados, inteligência artificial, bancos de dados, criptografia e teoria da computação.

Durante a disciplina foram estudados conceitos fundamentais para o desenvolvimento do raciocínio lógico e da resolução de problemas computacionais.

## 📖 Conteúdo abordado

- Lógica Proposicional
- Lógica de Predicados
- Tabelas-Verdade
- Equivalências Lógicas
- Quantificadores
- Teoria dos Conjuntos
- Relações
- Funções
- Princípio da Indução Matemática
- Técnicas de Contagem
- Permutações
- Arranjos
- Combinações
- Princípio da Inclusão e Exclusão
- Recorrências
- Grafos
- Árvores
- Álgebra Booleana

## 🎯 Objetivo

Este repositório tem como objetivo registrar a evolução dos estudos em Matemática Discreta, reunindo soluções para exercícios e materiais de apoio utilizados ao longo da disciplina.

Além de servir como registro acadêmico, o projeto também funciona como material de consulta para futuras disciplinas da área de Computação.
